# Events

# Try loot 
- Client > Client
- Used to trigger the TryLoot function
```lua
TriggerEvent('trolley:TryLoot')
```

# Try Destroy Loot
- Client > Client
- Used to trigger the TryDestroyLoot function
```lua
TriggerEvent('trolley:destroy')
```
